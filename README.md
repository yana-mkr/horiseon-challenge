A project to refractor code. This included meeting accessibility standards, changing semantic HTML elements and aligning them in a logical structure, adding accessible alt attributes, consolidating CSS and organizing to follow semantic structure, adjustment of HTML to fall in sequential order, and fixing and adding links.
